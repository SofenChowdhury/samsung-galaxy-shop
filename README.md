# samsung-galaxy-shop
SAMSUNG eCommerce app.
Create Next APP through command:
=> npx create-next-app
Than npm install cmd:
=> npm install --legacy-peer-deps
after that cmd:
=> npm install -g @sanity/cli
=> sanity upgrade
=> sanity init --coupon javascriptmastery2022
next cmd:
=> cd sanity-project-name
=> sanity start
